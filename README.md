# RepoDock

RepoDock is a lightweight macOS desktop app for discovering, organising, and launching local Git repositories.

The goal is to provide a fast, native alternative to manually hunting for repositories in Finder or Terminal.

## Early Goals
- Scan folders for Git repositories
- Display repositories with metadata
- Open repositories in VS Code
- Provide a clean, developer-first UI

## Status
Early development (v0.0.1)

## Tech Stack
- Swift
- SwiftUI
- AppKit (where needed)

## Roadmap
- Folder scanning
- Repo detection
- VS Code integration
- Git metadata
